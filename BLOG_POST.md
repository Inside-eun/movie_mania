# 예술영화 상영시간표, 어떻게 모아올까?

서울 곳곳에 흩어진 예술영화관들의 상영시간표를 한 곳에 모으는 건 생각보다 복잡한 일이다. 각 영화관 사이트를 일일이 방문해야 하고, 대형 멀티플렉스처럼 편한 앱도 없다. 이 사이트가 그 문제를 어떻게 해결하는지 정리해본다.

---

## 1. 데이터는 어디서 오는가

### 영화진흥위원회(KOBIS) API

국가에서 운영하는 영화 관련 공공 데이터다. 전국 모든 영화관의 상영 스케줄을 제공한다.

우리가 이 API에서 가져오는 건 두 가지:
- **박스오피스 순위**: 어제 기준 1~5위 영화 목록
- **극장별 상영 스케줄**: 특정 극장 코드로 조회하면 그 극장의 오늘 상영표가 나온다

### 한국영상자료원(KMDB) API

한국영상자료원 시네마테크KOFA의 상영 스케줄은 KOBIS에 없다. 별도로 KMDB API를 통해 가져온다.

---

## 2. 어떤 영화관을 조회하는가

서울 시내 29개 영화관을 조회 대상으로 지정해뒀다.

**독립/예술영화 전용관** (20곳)
- 씨네큐브광화문, 서울아트시네마, 인디스페이스, 필름포럼 등

**대형 체인 중 아트하우스관** (9곳)
- CGV 용산아이파크몰 19관
- CGV 압구정
- CGV 명동역 씨네라이브러리
- 롯데시네마 노원 6관 등

대형 체인의 경우, 아트하우스 전용 상영관만 골라서 가져온다. 예를 들어 CGV 용산아이파크몰은 19관(아트하우스) 상영작만, 나머지 상영관은 무시한다.

---

## 3. 데이터 처리 흐름

```
사용자가 날짜 선택 후 "검색" 클릭
         ↓
    캐시에 데이터 있나?
     ↙        ↘
   있음        없음
    ↓           ↓
캐시 반환    API 호출 시작
              ↓
        ┌─────────────────┐
        │ 박스오피스 조회  │  → 1~5위 영화 목록 확보
        └─────────────────┘
              ↓
        ┌─────────────────┐
        │ 29개 영화관     │
        │ 스케줄 병렬조회 │  → 8개씩 묶어서 동시 요청
        └─────────────────┘
              ↓
        ┌─────────────────┐
        │ 한국영상자료원  │  → KMDB API로 별도 조회
        │ 스케줄 조회     │
        └─────────────────┘
              ↓
        ┌─────────────────┐
        │ 필터링 & 정제   │
        │ - 박스오피스 제외│
        │ - 특정 상영관만 │
        └─────────────────┘
              ↓
        시간순 정렬 → 캐시 저장 → 화면 표시
```

---

## 4. 왜 박스오피스 영화를 제외하는가

예술영화관도 때로는 상업 대작을 튼다. "베놈"이나 "모아나"를 인디스페이스에서 상영할 수 있다. 하지만 이 사이트의 목적은 "예술영화를 찾는 것"이므로, 박스오피스 1~5위 영화는 결과에서 빼버린다.

---

## 5. 왜 특정 상영관만 필터링하는가

CGV 같은 대형 체인은 한 건물에 10개 넘는 상영관이 있다. 그 중 아트하우스로 운영되는 상영관은 보통 1~2개뿐이다.

| 영화관 | 아트하우스 상영관 |
|--------|-------------------|
| CGV 용산아이파크몰 | 19관 |
| CGV 대학로 | 5관 |
| CGV 강변 | 4관 |
| CGV 신촌아트레온 | 10관 |
| CGV 여의도 | 5관 |
| 롯데시네마 노원 | 6관 |
| 롯데시네마 신도림 | 4관 |

이 상영관 외의 스케줄은 전부 무시한다.

---

## 6. 캐시는 왜 필요한가

29개 영화관에 일일이 API 요청을 보내면 시간이 꽤 걸린다. 한 번 조회한 결과는 저장해뒀다가, 같은 날짜로 다시 조회하면 저장된 데이터를 바로 보여준다.

- 첫 조회: 30초~1분
- 캐시 사용 시: 즉시

---

## 7. 요약

1. 사용자가 날짜를 선택하면
2. 서버가 29개 영화관 + 한국영상자료원의 상영 스케줄을 API로 수집
3. 박스오피스 영화와 일반 상영관 데이터를 걸러내고
4. 시간순으로 정렬해서 보여준다

겉으로는 단순해 보이지만, 뒤에서는 공공 API 2종, 병렬 처리, 캐싱, 다단계 필터링이 돌아가고 있다.
